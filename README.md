##### **🏰** **Turn-Based Multiplayer Dungeon Crawler**

## 

Welcome to the **Turn-Based Multiplayer Dungeon Crawler** project! This game is built using **Unity 6** and **Netcode for GameObjects**, focusing on synchronized online gameplay and strategic turn-based mechanics.



###### **🎮 Game Overview**

A 2-4 player cooperative dungeon crawler where players navigate a shared grid, interact with the environment, and strategize to reach the exit.



###### **✨ Key Features**

\- 🔄 **Turn-Based Logic:** A strict turn-based system ensuring fair play and clear action sequences.

\- 🌐 **Server-Authoritative Networking:** Reliable multiplayer synchronization where the server validates all moves to prevent cheating.

\- 🗺️ **Dynamic Dungeon:** A grid-based map featuring walkable tiles, obstacles, and interactive objects.

\- ⚔️ **Interactive World:** Players can move, attack, and interact with doors, chests, and switches.



###### **🛠️ Technical Stack**

\- **Engine:** Unity 6

\- **Language:** C#

\- **Networking:** Unity Netcode for GameObjects (NGO)



###### **🏗️ Architecture**

The game utilizes a **Client-Server** model:

\- **Server:** Acts as the authority, processing rules, validating actions, and maintaining the game state.

\- **Client:** Handles user input, UI rendering, and displays the state synced from the server.
